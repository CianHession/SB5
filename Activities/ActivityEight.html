Learn how to Sew by hand

Sewing is an easy and fantastic thing to know and learn, you never know when it might come in handy!

Tips: This is a basic life skill that not many people know how to do. It takes a lot of practice to master but is a really useful skill to have. Learning this skill will allow you to repair tears in clothes so they can be used again. I think this is a good skill to learn as all you need is:

  • Needle
  • Thread
  • Material to work on
  • Scissors
  • Patience
  • Patience
  • Hand-eye coordination

Steps:

  • 1. Thread your needle and secure a knot:
    1. Start by cutting a length of thread
    2. Insert the thread in the loop on the needle and pull until the thread is doubled over
    3. Secure a knot at the ends of the thread by wrapping the thread around your index finger putting the thread ends inside the loop, then pull to make a knot
    4. Cut the end of the thread off with scissors just shy of the knot to tidy it up

  • 2. Start Stitching
  • Running Stitch - Use when stitching something quickly or to obtain a gathered appearance
    1. Insert the needle in the fabric in the direction you want to go and pull until tight. There is now 1 stitch in your fabric
    2. Insert the thread in the loop on the needle and pull until the thread is doubled over
    3. Repeat step 2 until you have reached your destination and pull until the fabric gets bunched up and tie off
  • Back Stitch - Use when you require a stronger stitch
    1. First do a normal stitch, then start your second stitch from where the first one ended. This stitch is stronger version of the running stitch
    2. Repeat and tie off
  • Blanket Stitch - Use when stitching two edges together
    1. For the first stitch - insert the needle in the same place, pass the needle through the loop and pull
    2. Insert the needle a small distance away from the first stitch. Pass the needle through the loop and pull
    3. Tie off when you are finished

  • 3. Tying Off:
    • On the side your free thread is exposed insert the needle into the previous stitch and pull until there is a small loop of thread, then put the needle through the loop and pull until tight. Repeat this step until secure

    Difficulty: Medium

    Duration: Don't be disheartened if your stitching looks messy, it takes time to develop this skill. KEEP PRACTICING!!
